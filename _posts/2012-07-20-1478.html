import re 
def grep(file,exp): 
    fd=open(file,'r') 
    lines=fd.read().splitlines() 
    fd.close()
    logs=[] 
    for line in lines: 
        if re.search(exp,line): 
            logs=logs.append(line) 
    return logs
这样是符合我需求了，大家可以按需扩展。
