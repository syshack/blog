为了增强用户体验，给留言板加入快捷提交。

查了很多资料，发现两种解决方案，现在将最简单的给大家共享下

上代码：

  • document.getElementById("comment").onkeydown = function (moz_ev)
  • {
  • var ev = null;
  • if (window.event){
  • ev = window.event;
  • }else{
  • ev = moz_ev;
  • }
  • if (ev != null && ev.ctrlKey && ev.keyCode == 13)
  • {
  • document.getElementById("submit").click();
  • }
  • }

    • 将上边js代码插入到comment.php 就大功告成了哈。
